# Symptom Tracking -- Competitive Analysis

## Bearable (Gold Standard, 4.8/5 iOS)
**LOVE:** Total customization, privacy-first (no ads, no data selling), replaces multiple apps, built BY patients, data export for doctor visits
**HATE:** App freezing/force quits mid-logging, paywall backlash (was free beta), overwhelming options (10+ min to log everything)
**WISH:** Adaptive UI that simplifies during flares, integrated community, wearable unification, better performance

## Flaredown (Community-First)
**LOVE:** Built-in community comparison with similar conditions, 200K+ food database, daily check-in habit building, free
**HATE:** Less frequent updates, less polished UI, hard to simplify during flares
**WISH:** More active development, rapid-logging mode for bad days

## CareClinic (Comprehensive, Aggressive Paywall)
**LOVE:** Interactive body maps, 24K+ drug database with interaction checking, AI import, correlation analysis engine
**HATE:** Can only track 3 symptoms free, constant subscription interruption on EVERY app open, feels extractive
**WISH:** Generous free tier to build habit before conversion

## Our Edge
- AI analysis via Intelligence Engine (no competitor has this)
- 34 medical API pipeline for cross-correlation
- Doctor Mode for appointment prep
- Zero paywall aggression
- All data feeds into unified correlation engine
