# Nutrition Tracking -- Competitive Analysis

## MyFitnessPal (200M+ users, largest database)
**LOVE:** 14M+ food database (including restaurants/chains), most integrations, barcode scanning
**HATE:** Full-screen video ads mid-logging (30 seconds), user-submitted bad data throws off macros, $20/mo paywall, 2018 data breach (150M users)
**WISH:** Verified database, ad-free logging, better free tier

## Cronometer (Accuracy Champion, 4.8/5 iOS)
**LOVE:** USDA-verified database, 40+ micronutrient tracking (vitamins, minerals others ignore), strong App Store ratings
**HATE:** Full-screen ads (same as MFP), feels like subscription bullying, $60/yr Gold tier
**WISH:** Ad-free without paywall, better macro focus

## MacroFactor (Research-Backed, Rising on Reddit)
**LOVE:** Adaptive algorithm adjusts macros weekly based on weight trends, fastest logging (10 vs 15 actions in MFP), science foundation, encourages consistency over perfection
**HATE:** No free version (only paid trial), no desktop version, no workout tracking
**WISH:** Free/freemium tier, desktop web app, workout integration

## MyNetDiary (Lanae's Current App)
**LOVE:** Clean UI, comprehensive nutrient tracking, meal-by-meal structure, good for logging individual foods
**HATE:** Smaller database than MFP, subscription for full features
**WISH:** Better food photo recognition, micronutrient depth like Cronometer

## Lose It (Simple Choice)
**LOVE:** Better UI than competitors, lower cost, calorie-focused simplicity
**HATE:** Less sophisticated than MacroFactor, database quality issues
**WISH:** Better database curation

- FODMAP/histamine/allergen classification
